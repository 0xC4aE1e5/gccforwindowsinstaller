How to install

Download 7-Zip (or any other extract app that can do .7z)

If you haven't already, download 7-Zip from http://www.7-zip.org/.

Extract the .7z that was downloaded

Open your downloads folder, and right click and click 7-Zip > Extract here

Extract the two files

Find mingw. 7-Zip > Extract Here. Find binutils. 7-Zip > Extract Here.

Add the bin folder to the path (optional)

If you'd like, add the bin folder to the path.

Delete all the 7z files (optional)

Find all of the .7z files and delete them.

Test it out

Go the bin folder and open a command prompt there. Create a hello world C script as test.c and run gcc test.c -o test.exe. You should now be able to run test.exe.
